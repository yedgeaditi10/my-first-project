# Weather App 🌦️

A web app that shows real-time weather using an API.

## 🚀 Features
- Search weather by city
- Shows temperature, humidity, wind
- Clean UI

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- OpenWeather API

## 📦 Installation
git clone https://github.com/username/weather-app.git
cd weather-app
npm install

## ▶️ Usage
npm start
